Projet en node JS 
Pour le lancer faire les commandes dans allocinoch:
cd client
npm install
npm start

Puis dans server:
cd server
npm install
npm start

Vous n'avez plus qu'a lancer http://localhost:8080 quand les téléchargements sont fini
